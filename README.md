## Install

Copy this folder to:

```text
~/.codex/pets/rei-blue-pilot
```

Make sure these two files are directly inside the folder:

```text
pet.json
spritesheet.webp
```

Then open Codex:

1. Go to Settings > Appearance > Pets.
2. Click Refresh.
3. Select Blue Pilot.
4. Use `/pet` or Wake Pet if the desktop pet is tucked away.
